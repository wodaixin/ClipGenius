# Getting Started

Three paths depending on your goal:

| Path | Who | Time | What you'll do |
|---|---|---|---|
| **End User** | Anyone using ClipGenius | ~5 min | Install, configure Firebase, start capturing |
| **Self-Hosted Developer** | Deploying your own instance | ~10 min | Everything in End User path + AI keys + env config |
| **Contributor** | Adding features to the project | ~15 min | Everything in Self-Hosted path + local dev setup |

## Prerequisites

- **Node.js** 18 or later (`node -v`)
- **npm** 9 or later (`npm -v`)
- **Firebase account** — required for cloud sync (free tier is sufficient)
- **Google AI Studio account** — required for AI features (free tier available)
- For contributors: a GitHub account with push access or a fork
