# 快速开始

5 分钟内完成首次剪贴板捕获。

## 第一步：启动应用

```bash
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000)，你应该能看到粘贴区域。

## 第二步：捕获内容

在输入框以外的区域按 `Cmd/Ctrl+V` 即可捕获剪贴板内容。ClipGenius 会自动识别内容类型：

| 你复制的内容 | ClipGenius 识别为 |
|---|---|
| 浏览器中的图片 | `image` |
| Facebook 视频 | `video` |
| 网页链接 | `url` |
| Markdown 文本 | `markdown` |
| 源代码 | `code` |
| 纯文本 | `text` |

## 第三步：探索功能

### 预览和编辑

点击历史面板中的任意卡片可预览。再次点击名称或摘要可直接编辑。

### 搜索

使用历史面板顶部的搜索栏，可按名称、摘要或内容搜索。支持模糊搜索。

### 置顶项目

点击卡片上的图钉图标将其置顶。置顶项目会出现在顶部，且不会被「清除未置顶」操作删除。

### AI 聊天

点击任意卡片上的聊天图标，以该内容为上下文打开 AI 聊天功能。可就内容提问。

### AI 图片生成

点击 **Generate Image** 打开图片生成弹窗，描述你想要生成的内容。

### 设置

点击粘贴区顶部的设置图标，可以配置：
- AI 提供商选择（Gemini / Minimax）
- 按内容类型的 AI 路由配置
- API 密钥和模型覆盖
